# Agent Visualizer - 产品需求文档

## 1. 产品概述

Agent Visualizer 是一个可视化工具，用于展示 AI Agent 之间的工作流程和协作关系。帮助开发者、团队负责人和研究人员理解 Agent 的交互模式。

## 2. 目标用户

- AI 开发者
- 团队负责人
- 研究人员
- 对 Agent 工作原理感兴趣的爱好者

## 3. 核心功能

### 3.1 Agent 列表面板
- 显示所有 Agent 的名称和状态
- 状态指示器：运行中(绿色)、等待中(黄色)、已完成(灰色)、失败(红色)
- 点击查看 Agent 详细信息

### 3.2 工作流程可视化
- 节点式流程图展示 Agent 协作关系
- 节点：代表单个 Agent
- 连线：代表消息传递方向
- 动画效果：消息在连线上流动

### 3.3 时间线视图
- 按时间顺序展示所有事件
- 支持拖拽浏览历史
- 显示每个事件的详细信息

### 3.4 控制面板
- 播放/暂停动画
- 快进/快退
- 重置视图
- 导出为 PNG/JSON

### 3.5 消息详情
- 点击节点查看消息历史
- 显示发送者、接收者、内容、时间戳

## 4. UI 设计

### 布局
- 左侧：Agent 列表面板 (250px)
- 中间：可视化流程图 (flex: 1)
- 右侧：详细信息面板 (300px，可折叠)
- 顶部：控制栏 (60px)

### 视觉风格
- 深色主题 + 渐变强调色
- 流畅的动画效果
- 清晰的状态颜色编码
- 现代卡片式设计

### 颜色方案
- 背景：#0f172a (深蓝黑)
- 面板背景：#1e293b
- 边框：#334155
- 主色调：#8b5cf6 (紫色)
- 成功：#22c55e (绿色)
- 进行中：#eab308 (黄色)
- 失败：#ef4444 (红色)
- 等待：#6b7280 (灰色)

## 5. 技术实现

- 纯前端 Web 应用
- 使用原生 JavaScript + CSS
- SVG 用于流程图绘制
- 不依赖外部框架，保持轻量
